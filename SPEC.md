# 楽器練習記録・上達トラッカー【無料】無料Webツール - 技術仕様書

## 概要

**サービス名**: Music Practice Log
**バージョン**: 1.2.2
**更新日**: 2026-06-10
**URL**: https://appadaycreator.com/music-practice-log/

楽器種類・練習内容・時間を記録。上達の軌跡を可視化してモチベーション維持に。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- Chart.js（グラフ描画）
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. 日付を選択する（デフォルトは本日）
2. 楽器・練習内容・練習時間を入力する
3. 「記録する」ボタンをクリックする
4. 記録一覧で過去10件のデータを確認する
5. グラフで過去30日間の練習時間の推移を可視化
6. 楽器別集計で各楽器の練習時間を確認
7. 記録の削除ボタン（🗑️）で間違った記録を削除
8. 継続的に記録して変化を追跡する

## よくある質問（FAQ）

**Q: 楽器練習記録・上達トラッカーは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 記録データはどこに保存されますか？**

ご使用のブラウザのローカルストレージに保存されます。他のデバイスとは共有されません。

**Q: ブラウザを閉じても記録は残りますか？**

はい、同じブラウザを使う限りデータは保持されます。

**Q: 記録を削除するにはどうすればいいですか？**

記録一覧の各行にある削除ボタン（🗑️）をクリックすると、その記録を削除できます。確認ダイアログが表示されます。全削除はブラウザのサイトデータをクリアしてください。

**Q: グラフは何を表示していますか？**

過去30日間の日別練習時間を楽器別に色分けした折れ線グラフです。複数の楽器を練習している場合、各楽器の進捗が一目でわかります。

**Q: 過去の日付を記録できますか？**

はい。日付フォームで過去の日付を選択して記録できます。これにより、うっかり記録し忘れた日の練習時間を後から追加できます。

**Q: スマートフォンで利用できますか？**

はい、スマートフォン・タブレット・PCすべてでご利用いただけます。
